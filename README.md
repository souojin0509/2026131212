# 2026131212
'나의 공부 성향 테스트'
SW프로그래밍의 기초 기말 대체 과제로 제작한 웹 기반 심리테스트입니다.

# 프로젝트 소개
사용자가 8개의 질문에 답하면 자신의 공부 성향을 분석하여 결과를 제공하는 웹페이지입니다.
총 4가지 유형으로 결과가 분류됩니다.
- 계획형 전략가
- 벼락치기 생존형
- 감각형 몰입러
- 완벽주의 정리형

# 주요 기능
- 객관식 심리테스트 진행
- 답변 기반 성향 분석
- 유형별 설명 제공
- 공부 팁 제공
- 추천 공부 장소 제공

# 사용 방법
1. 테스트 시작하기 버튼을 누릅니다.
2. 8개의 질문에 답합니다.
3. 결과 보기 버튼을 통해 자신의 공부 성향을 확인합니다.
